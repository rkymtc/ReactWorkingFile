## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/reactworkingfile
```

Proje dizinine gidin

```bash
  cd reactworkingfile
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm start
```

  
## Kullanılan Teknolojiler

- React
- Bootstrap
