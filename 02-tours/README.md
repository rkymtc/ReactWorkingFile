# React Project

2. [Tours](https://react-working-file-2.netlify.app/)

## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/rkymtc/reactworkingfile
```

Proje dizinine gidin

```bash
  cd reactworkingfile
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm start
```

  
## Kullanılan Teknolojiler

- CSS
- React
